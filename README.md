*Installation*

Clone the repository:

```bash
git clone https://github.com/Kipo77/JBM-XMD.git
cd JBM-XMD
```

Install dependencies:

For *Node.js*:

```bash
npm install
```

For *Python*:

```bash
pip install -r requirements.txt
```

---

*Usage*

To run the project:

For *Node.js*:

```bash
npm start
```

For *Python*:

```bash
python app.py
```
